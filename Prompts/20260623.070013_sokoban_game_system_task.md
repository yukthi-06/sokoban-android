# 🧩 CORE REQUIREMENTS

## 1. Game Engine (PURE JAVA MODULE — NO ANDROID CODE)

Create a reusable module:

```
sokoban-engine/
```

### Must include:

### A. Model

* Grid representation
* Player position
* Box positions
* Goal positions

### B. Parser

Parse XML and extract:

* Width, Height
* GridWithBlankSpaces
* Convert into internal grid

Handle symbols:

| Symbol | Meaning           |
| ------ | ----------------- |
| `#`    | wall              |
| `$`    | box               |
| `@`    | player            |
| `.`    | goal              |
| space  | empty             |
| `/`    | newline separator |

Also handle combined cases like:

* box on goal
* player on goal (if encoded later)

---

### C. Game Rules Engine

Implement:

* move up/down/left/right
* push box rules
* block movement on walls
* win condition: all boxes on goals

Provide:


### D. No Android dependencies

Engine must be usable in:

* CLI tests
