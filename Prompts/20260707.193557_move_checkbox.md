# User Request
> the checkbox should be in the settings screen.
> The label should be 'Show completed levels'

# Implementation Details

The checkbox for showing/hiding completed levels has been migrated from the main dashboard to the Settings screen.

1. **Settings Screen Updates (`activity_settings.xml`, `SettingsActivity.java`)**:
   - Added a new `LinearLayout` to hold the "Show completed levels" label and its `CheckBox`.
   - The checkbox is placed visually on the right side of the text inside the `LinearLayout`.
   - Updated `SettingsActivity.java` to load, display, and persist the new `show_completed` preference (defaults to `true`).

2. **Dashboard Updates (`activity_main.xml`, `MainActivity.java`)**:
   - Removed the inline `filterLayout` (which contained the previous checkbox) from the main UI.
   - Restored original margin constraints for the `RecyclerView`.
   - Updated the filtering logic in `MainActivity.java` to read the inverted boolean state `show_completed`. If `showCompleted` is false, it skips adding completed levels to the adapter list. The index mapping continues to work smoothly.
