# User Request
> change package from com.sokoban to com.vypeensoft.sokoban

# Goal Description

The goal is to change the base package name of the application from `com.sokoban` to `com.vypeensoft.sokoban`. This involves updating package declarations in all Java classes, updating layout XML files that reference custom views, updating the `build.gradle` scripts for `applicationId` and `namespace`, and restructuring the physical directories to match the new package paths.

## Proposed Changes

1. **Move Directories:**
   Move all contents from `com/sokoban` to `com/vypeensoft/sokoban` in the following source sets:
   - `sokoban-android/src/main/java/`
   - `sokoban-engine/src/main/java/`
   - `sokoban-engine/src/test/java/`

2. **Update Source Files:**
   - Update `package` declarations and `import` statements in all Java files.
   - Update XML references in `sokoban-android/src/main/res/layout/activity_game.xml` to point to `com.vypeensoft.sokoban.android.view.GameView`.

3. **Update Build Configurations:**
   - Modify `sokoban-android/build.gradle` to change `namespace 'com.sokoban.android'` to `namespace 'com.vypeensoft.sokoban.android'`.
   - Modify `sokoban-android/build.gradle` to change `applicationId "com.sokoban.android"` to `applicationId "com.vypeensoft.sokoban.android"`.
