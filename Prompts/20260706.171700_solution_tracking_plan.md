# Solution Recording & Dashboard Progress Plan

The goal is to track the sequence of moves and pushes during gameplay, save it to a JSON solution file upon completion, and reflect this progress visually on the dashboard grid.

## User Review Required

- **Storage Location**: I will create a new directory `/sdcard/Vypeensoft/Sokoban/solutions/` to save the `[GAME_NUMBER]_solution.json` files. This keeps the `levels` folder clean. Is this acceptable?
- **Visual Indicator**: I plan to change the level number text color to a bright, vibrant green (`#4CAF50`) on the dashboard when a level is completed. Is this enough, or would you strictly prefer a tick mark icon?

## Proposed Changes

### 1. Data Model (`GameState.java` & `GameEngine.java`)
- **[MODIFY] GameState.java**: Add a `String moveSequence` field to store the sequence of moves.
- **[MODIFY] GameEngine.java**: 
  - Update the `move` method. When a move occurs, append a lowercase character (`u`, `d`, `l`, `r`).
  - When a push occurs, append an uppercase character (`U`, `D`, `L`, `R`).

### 2. Gameplay Tracking & Saving (`GameActivity.java`)
- **[MODIFY] GameActivity.java**:
  - Track `startTime` when the level loads.
  - When the game is won, calculate `timeTaken = (System.currentTimeMillis() - startTime) / 1000`.
  - Format the level number exactly as we do for the dashboard (e.g. `007.json` -> `7`).
  - Write a JSON object to `/sdcard/Vypeensoft/Sokoban/solutions/7_solution.json` containing:
    ```json
    {
      "moves": <movesCount>,
      "pushes": <pushesCount>,
      "timetaken": <seconds>,
      "sequence": "<moveSequence>"
    }
    ```

### 3. Dashboard UI (`MainActivity.java`, `LevelAdapter.java`, `item_level.xml`)
- **[MODIFY] item_level.xml**: Ensure we have a default accent color, but make it easily swappable programmatically.
- **[MODIFY] MainActivity.java**: 
  - On `onResume`, reload the list of completed levels so that returning from a victory updates the dashboard immediately.
- **[MODIFY] LevelAdapter.java**:
  - Add a boolean flag or a `Set<String>` of completed level numbers.
  - In `onBindViewHolder`, if the current level number is in the completed set, change the `levelNumberText` color to green.
