# User Request
> Have a checkbox for 'Show/Hide completed levels in browsing'
> Also the checkbox placement must be on right side of the text

# Implementation Details

To allow users to filter out completed levels from the level browser, the following changes were made:

1. **Dashboard UI Update (`activity_main.xml`)**: 
   - Added a new `LinearLayout` immediately below the "Select Level" subtitle.
   - Inside this layout, placed a `TextView` saying "Show/Hide completed levels" followed by a `CheckBox` widget. By placing the `CheckBox` after the `TextView` in the horizontal layout, the checkbox visually appears to the right of the text.

2. **Dashboard Logic (`MainActivity.java`)**:
   - Initialized and bound the new `checkHideCompleted` checkbox.
   - Its state is read from and saved to `SokobanPrefs` under the key `hide_completed`.
   - Updated the `initializeDashboard` method to filter the `levelFiles` list before passing it to the adapter.
   - **Crucial fix**: Mapped the filtered adapter positions back to their original `currentLevelIndex` values so that tapping a level in the filtered view opens the correct target level in the game.
