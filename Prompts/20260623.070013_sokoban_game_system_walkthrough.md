### B. Parser

* Width, Height
* GridWithBlankSpaces

Handle symbols:

| Symbol | Meaning           |
| ------ | ----------------- |

| `$`    | box               |
| `@`    | player            |
| `.`    | goal              |
| space  | empty             |
| `/`    | newline separator |

Also handle combined cases like:

* box on goal
* player on goal (if encoded later)
