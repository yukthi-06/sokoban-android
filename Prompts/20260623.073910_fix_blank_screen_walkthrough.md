# User Prompt

> the game screen is blank

# Walkthrough

The game screen was appearing blank because the custom `GameView` canvas had its layout height and width collapsed to 0. This layout collapse was caused by a syntax error in the top header card (`MaterialCardView`) inside `activity_main.xml`:

```xml
<com.google.android.material.card.MaterialCardView
    id="@+id/headerCard"
```

Because the `id` was declared without the `android:` namespace prefix (`android:id`), the Android layout compiler did not bind the ID properly to the view. As a result, the constraints referencing the headerCard (`app:layout_constraintTop_toBottomOf="@id/headerCard"`) failed to resolve, collapsing the custom `GameView`.
